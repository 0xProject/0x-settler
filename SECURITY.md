# Security Policy

## Reporting a Vulnerability

0x hosts a bug bounty program on Immunefi at https://immunefi.com/bug-bounty/0x

If you have found a vulnerability in our project, it must be submitted through Immunefi's platform. Immunefi will handle bug bounty communications.

See the bounty page at Immunefi for more details on accepted vulnerabilities, payout amounts, and rules of participation.

## Supported Versions

Only the latest deployed version of Settler is supported. Use the deployer/registry contract to find the current active address.
